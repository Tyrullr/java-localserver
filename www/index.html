Java custom Web Server

A high-performance HTTP/1.1 Non-Blocking server built entirely in Java Core.

Server Capabilities

  • Single-threaded event loop (`java.nio.channels.Selector`)
  • Manual parsing of chunked & unchunked requests
  • Support for GET, POST, DELETE methods
  • Stateful Session Management via secure Cookies
  • Asynchronous multi-process Python CGI scripts
  • Directory listings & custom error page mapping
